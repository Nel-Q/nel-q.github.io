I Present to you Flixster:

Flixster is an android app that allows you to view trending movies with the title, poster as well as an overview for each movie. The views are responsive both in portrait and landscape mode

The app was later updated to include trailers as well as ratings for the different movies

The app was made using two external libraries:

  • Android Async HTTP: Simple asynchronous HTTP requests with JSON parsing
    •
  • Glide: Image loading and caching library for Android
